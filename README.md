# Video Background React App

A minimal React + Vite app featuring a full-screen background video with a tasteful overlay and a glassy content card.

## Quick start

- Prerequisites: Node.js 18+ and npm.
- Install deps: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build` then `npm run preview`

## Customize

- Change headline/subtitle: edit `src/App.jsx`.
- Update colors and look: tweak CSS variables in `src/styles.css`.
- Swap the video: update `VIDEO_SRC` in `src/App.jsx`.

## Notes

- The video is muted, looping, and plays inline to satisfy browser autoplay policies.
- Fonts load from Google Fonts; you can remove or replace them in `index.html`.
